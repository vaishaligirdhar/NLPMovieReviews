nlp_movie_reviews.py
This fils is the python script using NLTK's movie_reviews corpus for training.
Please note that this is the same corpus as polarity dataset v2.0 from 
http://www.cs.cornell.edu/People/pabo/movie-review-data/
4/5 of the data is used for training the model and 1/5 for testing

nlp_movie_reviews_own_corpus.py
This fils is the python script using user-loaded corpus for training.
The corpus used is the polarity dataset v2.0 from 
http://www.cs.cornell.edu/People/pabo/movie-review-data/
4/5 of the data is used for training the model and 1/5 for testing

Run the program:
open cmd.exe, traverse to directory containing the python files,
then run the following commands
--> python emos_ai_test.py
or
--> python emos_ai_test_own corpus.py

Program output:
training sample and test sample instances
accuracy of text classification
most informative features

NOTE: please ensure that the python files and the database are in the same folder
for emos_ai_test_ownCorpus.py
